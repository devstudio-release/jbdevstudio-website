There are two versions of Red Hat JBoss Developer Studio 4.1 available:

1) Red Hat JBoss Developer Studio bundled as an installer for your platform*

2) Red Hat JBoss Developer Studio bundled with EAP 5.1 as an installer for your platform*.

To download, you must be an existing EAP or Red Hat JBoss Developer Studio customer.

Check the Customer Portal for more information.

* - Windows 32-bit, Linux 32- or 64-bit, Mac OSX 32-bit
