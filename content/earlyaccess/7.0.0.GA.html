Installation :: Core Tooling

1. If you don't already have one installed, you will need to download and install Sun/Oracle or Open JDK 7 (or 6).

2. Download the installer for your platform.

3. Double-click the installer file to run it, or open a terminal and type java -jar /path/to/jbdevstudio-*.jar
