Red Hat JBoss Developer Studio 4.x - Updates

How Do I Install?


  1. Start JBDS. Go to Help > Install New Software... > Add...

  2. Paste in the URL of the update site(s) you've chosen from the list at left. Browse the site for updates you'd like to install. Restart when prompted.

This site contains the latest released updates for Red Hat JBoss Developer Studio. For more information, see
http://www.redhat.com/products/jbossenterprisemiddleware/developer-studio/.

If you do not already have a copy of Red Hat JBoss Developer Studio, click here to purchase or download it free.

Point your Red Hat JBoss Developer Studio at one or more of the following sites:

  • Red Hat JBoss Developer Studio 4.x Core Tools & Platform Updates (From Eclipse.org & other vendors)
    • https://devstudio.jboss.com/updates/4.0/

  • Red Hat JBoss Developer Studio 4.x 'Extras' - Third-Party Extensions & Tools
    • https://devstudio.jboss.com/updates/4.0/extras/

  • Red Hat JBoss Developer Studio 4.x 'Technology Preview' - Additional Extensions & Tools
    • https://devstudio.jboss.com/updates/4.0/techpreview/

Note: If you install Red Hat JBoss Developer Studio updates into an existing Eclipse 3.6.2 (Helios SR2) installation, you will most likely need to tweak your eclipse.ini file to give Eclipse more memory than is allocated by default. This should work for you but is currently Unsupported. Please see the following tips:
