Red Hat JBoss Developer Studio 4.x - Updates

How Do I Install?


  1. Start JBDS as normal.

  2. Go to Help > Install New Software... > Add...

  3. Paste in the URL of the update site(s) you've chosen from the list at left.

  4. Browse the site for updates you'd like to install, select them, then click through the dialogs to accept licenses and install the new software.

  5. Restart JBDS when prompted.
