Red Hat JBoss Developer Studio 8.x - Central - Integration Stack


This folder contains a composite site which allow a user to resolve all upstream dependencies of a Red Hat JBoss Developer Studio JBoss Central connector via a single URL. Simply launch JBoss Developer Studio, open JBoss Central, then select the Software/Update tab to explore this site for installable connectors.

Content available from JBoss Central has been tested by Red Hat but support of the content varies depending on content type. Click here for details.
