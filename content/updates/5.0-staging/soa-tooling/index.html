Red Hat JBoss Developer Studio 5.x - SOA Tooling

How Do I Install?


  1. Start JBDS. Go to Help > Install New Software... > Add...

  2. Paste in the URL of the update site at left. Browse the site for updates you'd like to install. Restart when prompted.

Latest Build:    5.0.6.GA

This site contains the latest stable milestone updates for Red Hat JBoss Developer Studio 5.x - Early Access.
